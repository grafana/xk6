Grafana **xk6** `v1.3.7` is here! 🎉

This release switches the Docker base image to Chainguard to eliminate CVE vulnerabilities and significantly reduce ongoing maintenance burden ([#439](https://github.com/grafana/xk6/pull/439)).

  - Migrated from `golang:alpine` to `cgr.dev/chainguard/go:latest-dev` 
  - Achieves **zero known CVEs** in the base image
  - Eliminates the continuous maintenance overhead of patching Alpine/Golang CVEs
  - Updated Go to version 1.26.x via Chainguard's latest stream
  - Security tools (gosec, govulncheck) are now installed from source during build
