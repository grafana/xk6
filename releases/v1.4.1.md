Grafana **xk6** `v1.4.1` is here! 🎉

This release focuses on bug fixes and maintenance updates, particularly for k6 v2 compatibility and extension validation workflows.

## Bug fixes

- **xk6 lint build check for k6 v2 extensions** ([#491](https://github.com/grafana/xk6/pull/491)): fixed the `lint` build check to resolve the correct k6 version for local extensions, so `xk6 lint` works properly with extensions using k6 v2.
- **Extension validation no longer blocked by security checks** ([#490](https://github.com/grafana/xk6/pull/490)): removed the `security` job dependency from the `lint` and `smoke` jobs in the extension validation workflow, so those checks still run even if the security job fails.

## Dependency updates

- Updated `github.com/masterminds/semver/v3` to `v3.5.0` ([#495](https://github.com/grafana/xk6/pull/495))
- Updated `github.com/mattn/go-isatty` to `v0.0.22` ([#493](https://github.com/grafana/xk6/pull/493))
- Updated `gosec` to `v2.26.1` in CI workflows and the devcontainer ([#494](https://github.com/grafana/xk6/pull/494), [#492](https://github.com/grafana/xk6/pull/492))
- Various CI and tooling dependency updates ([#489](https://github.com/grafana/xk6/pull/489), [#488](https://github.com/grafana/xk6/pull/488), [#487](https://github.com/grafana/xk6/pull/487), [#486](https://github.com/grafana/xk6/pull/486))
