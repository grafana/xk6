Grafana **xk6** `v1.3.6` is here! 🎉

This is a security patch release that updates Go to version 1.25.8 and addresses multiple security vulnerabilities.

## Security

- Update Go to 1.25.8 and Docker base images to fix CVEs ([#435](https://github.com/grafana/xk6/pull/435))
  - Note: CVE-2026-27171 remains unresolved as no fix is available yet in the golang base image

- Fix path traversal vulnerabilities in scaffold module ([#435](https://github.com/grafana/xk6/pull/435)):
  - G703 (CWE): Path traversal via taint analysis  
  - G122 (CWE-367): TOCTOU race condition in filesystem operations
  - **Solution**: Replaced `filepath.Walk` with `os.Root` API for built-in security guarantees

- Update github.com/cloudflare/circl from 1.6.1 to 1.6.3 ([#433](https://github.com/grafana/xk6/pull/433))

## Docker

- Update Go version to 1.25.8-alpine3.23 in Dockerfiles ([#435](https://github.com/grafana/xk6/pull/435))
