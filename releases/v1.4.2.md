Grafana **xk6** `v1.4.2` is here! 🎉

This release focuses on security updates and dependency upgrades.

## Security fixes

- Pinned `go` toolchain to latest patch version for security ([#509](https://github.com/grafana/xk6/pull/509), [#510](https://github.com/grafana/xk6/pull/510))
- Updated `golang.org/x/net` to `v0.53.0` ([#502](https://github.com/grafana/xk6/pull/502))
- Updated `github.com/go-git/go-git/v5` to `v5.19.0` ([#507](https://github.com/grafana/xk6/pull/507))

## Dependency updates

- Updated `golang.org/x/mod` to `v0.36.0` ([#508](https://github.com/grafana/xk6/pull/508))
- Updated `github.com/ctrf-io/go-ctrf-json-reporter` to `v0.1.0` ([#504](https://github.com/grafana/xk6/pull/504))
- Various CI, GitHub actions, and Docker base image updates ([#498](https://github.com/grafana/xk6/pull/498), [#499](https://github.com/grafana/xk6/pull/499), [#500](https://github.com/grafana/xk6/pull/500), [#501](https://github.com/grafana/xk6/pull/501), [#503](https://github.com/grafana/xk6/pull/503), [#511](https://github.com/grafana/xk6/pull/511), [#512](https://github.com/grafana/xk6/pull/512), [#513](https://github.com/grafana/xk6/pull/513), [#514](https://github.com/grafana/xk6/pull/514))
